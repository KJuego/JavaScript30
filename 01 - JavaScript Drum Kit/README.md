# JavaScript Drum Kit Challenge | KJuego

## What does it do?
    This is a simple drum kit that associates one of nine sounds with the ASDFGHJKL keys on your keyboard.

## How can you use it?

1. Download all of the files in this folder (01 - JavaScript Drum Kit).
2. Open the index-KJuego.html file (or the index-WesBos.html) in a browser.
3. Press any of the ASDFGHJKL keys and jam out!
